## 注释

定义: 对代码进行解释说明的文字     

**注释不要嵌套**    

1. 单行注释: `//`  
2. 多行注释: `/* */`  
3. 文档注释: `/*** ***/`  

范例: 

```java
public class Main {
    //main方法,表示程序的主入口
    public static void main(String[] args) {
        //输出语句(打印语句)
        System.out.println("Hello and welcome!"); 
    }
}
```

定义: 被赋予特定含义的英文单词

class用于创建/定义一个类
